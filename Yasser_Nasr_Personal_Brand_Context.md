## 2. Brand Foundation

### Purpose
To bridge the gap between brands and their audiences through clear, meaningful, and well-crafted visual communication.

### Mission
To turn ideas into visual systems, digital experiences, and communication that people can understand, feel, and remember.

### Vision
To become a recognized senior designer and creative partner for brands across GCC and international markets, with a long-term path toward a personal creative studio.

Many businesses have disconnected brand elements: unclear messaging, inconsistent visuals, weak digital presence, and communication that does not reach the audience clearly. Yasser's role is to transform that confusion into a connected visual system across brand identity, websites, campaigns, social content, motion, print, and presentations.

### What Yasser should become known for
Building brand and digital experiences from the smallest details to the complete system.

## 5. Brand Personality

The brand personality is:

**Strategic - Modern - Refined - Expressive - Reliable**

### Strategic
Every design decision should have a reason. The work should connect to message, audience, business need, and communication purpose.

### Modern
The brand should feel current, digital, and relevant to contemporary creative teams and international companies.

### Refined
The identity should feel clean, controlled, detailed, and carefully crafted. It should not feel noisy or overloaded.

### Expressive
The brand should still show artistic character, feeling, and creative personality through the pixel mascot, pixel language, and visual storytelling.

### Reliable
The brand must remain suitable for recruiters, agencies, companies, and premium clients. It should feel creative without losing professional trust.

## 7. Brand Principles

1. Start with the message before the visuals.
2. Build systems, not isolated designs.
3. Use detail to create stronger perception.
4. Keep the identity clean so the work can speak.
5. Balance creativity with professional clarity.
6. Make every visual touchpoint feel connected.
7. Treat the pixel as a concept, not a decoration.
8. Let the portfolio work bring color and energy.

## 8. Visual Identity Concept

### Main idea
Every digital visual experience begins with a pixel.

The pixel represents the smallest unit of visual communication: the place where detail, structure, and meaning begin. The identity uses the pixel as a metaphor for Yasser's design mindset: building from precise details toward complete brand and digital experiences.

### Concept statement
Every great visual starts with a pixel. From the smallest detail to the complete system, Yasser Nasr builds brand and digital experiences that help businesses communicate clearly and connect with their audiences.

### Visual direction
Dark-first editorial identity with refined pixel character.

The identity should be creative, memorable, artistic, and personal - but still refined enough for senior design roles, international companies, recruiters, and future premium clients.

The mascot is a signature identity asset. It expresses artistic character, memorability, and the pixel concept. It should appear with intention, not everywhere.

### Mascot usage rules
Use the mascot in:
- Hero section
- About section
- Loading animation
- Favicon
- Social avatar
- Footer signature
- Selected identity moments

Avoid using the mascot in:
- Every project card
- Every section
- Repetitive backgrounds
- Every headline
- Very small areas where detail is lost

## 10. Typography System

### Display font
Alpharush Pixel Font

Use for:
- Logo / wordmark
- Big short headlines
- Section labels
- Numbers
- Tags
- Loading screen
- Identity statements

### Body font
Poppins

Use for:
- Paragraphs
- Case studies
- Services copy
- Navigation
- Forms
- Project descriptions
- Long-form website content

### Typography rule
Pixel font creates identity. Poppins creates readability.

Do not use the pixel font for long paragraphs or small body copy.

## 11. Color System

Use grayscale only, without true black or true white.

| Role | Color |
|---|---|
| Main Dark Background | #1D1D1B |
| Deep Surface | #242424 |
| Card Surface | #2F2F2D |
| Border / Divider | #3A3A38 |
| Main Text | #F2F0EA |
| Secondary Text | #B8B8B2 |
| Muted Text | #7A7A76 |
| Light Background | #E8E5DD |
| Dark Text on Light | #151515 |

### Color rule
The brand stays grayscale. The work brings the color.

This keeps the identity clean and lets portfolio projects become the visual proof.

---

## 12. Pixel Usage Rules

Use pixel language as:
- Grid logic
- Motion reveal
- Hover effects
- Section labels
- Small graphic details
- Loading animation
- Subtle texture
- Project transitions

Do not overload the identity with pixel art. The pixel should feel like a refined system, not a retro or gaming theme.

### Balance rule
80% refined editorial design. 20% pixel character.

## 21. Claude Code / Website Build Instructions

When building the website, follow these rules:

- Use a dark-first editorial design system.
- Use grayscale colors from the approved palette.
- Use the pixel mascot as a signature identity asset.
- Use Alpharush Pixel Font for display moments only.
- Use Poppins for body copy and interface text.
- Do not overuse pixel art.
- Use large spacing, clean layout, and strong typography.
- Keep the website professional enough for recruiters and international companies.
- Make the work/project visuals the main source of color.
- Build reusable components for project cards, sections, services, and case studies.
- Prioritize mobile readability and performance.
- Make the website SEO-friendly for Senior Graphic Designer, Multimedia Designer, Brand Designer, Visual Designer, and Digital Designer searches.

Recommended stack:
- TypeScript
- Next.js
- React
- Tailwind CSS
- GitHub
- Vercel

## 22. Do / Don't Rules

### Do
- Lead with Senior Graphic Designer for search and recruitment.
- Support with Brand & Digital Designer for broader value.
- Use the pixel concept as a strategic metaphor.
- Keep layouts refined, dark-first, and editorial.
- Let the mascot create memorability.
- Show thinking, process, and results.
- Write clearly and confidently.
- Build systems, not isolated visuals.

### Don't
- Do not position Yasser as only a social media designer.
- Do not make the identity feel like gaming or retro only.
- Do not overuse the mascot.
- Do not use pixel font for long body text.
- Do not make the brand feel like an agency before it is ready.
- Do not claim metrics that are not true.
- Do not make the portfolio just a visual gallery.
- Do not use generic creative buzzwords without proof.
